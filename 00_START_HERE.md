## üí° Tips & Tricks

### Quick Edits
- Edit HTML with Notepad (built-in to Windows)
- Or use: VS Code (free), Sublime Text, or similar
- Changes apply immediately when you refresh browser

### Mobile Viewing
- Press F12 in browser
- Click mobile device icon
- Test on different screen sizes

### Color Customization
- Use: https://colorpicker.com
- Pick your color
- Copy hex code (e.g., #FF0000)
- Find & replace in styles.css

### Image Optimization
- Compress: https://tinypng.com
- Keep: Under 200KB per image
- Format: JPG for photos, PNG for logo
